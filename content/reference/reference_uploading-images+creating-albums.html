All photos should be uploaded to Flickr, which is organized via albums. Any new "activity" needs a new album which must be created. Sort all photos into the appropriate album, either to an existing one or to a new created album.


NOTE: All files must be renamed as described in the File Naming Conventions.

Flickr upload tutorial in Google Drive


1) First, navigate to Flickr via the Photo Archive link in the navigation bar


2) Log into Flickr with the login: makingknowing; password: Chandler260


3) Click on "upload" and choose the files you want to upload
*Alternately, you can download and install the Flickr Uploadr program for your computer (but this is not necessary)



4) Return to the main Photostream page and click on "Albums"


5) Create a new album by clicking on "Add New Album"

6) Name this album:
  • (FA/SP)YY_description_initials
  • For example, FA16_Dyeing Yarn_MM NR



7) Drag and drop the photos that you previously uploaded into the new album from the picture bar at the bottom


8) You can also add a brief description as a comment to your photos if you feel that additional information beyond the title will be helpful. Finally, click "save."
