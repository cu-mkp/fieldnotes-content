Batch Renaming of Files

If you would like to rename many files all at once without having to edit each photo individually, there are programs that will do this for you. Rename the files BEFORE uploading to either the Wiki or Flickr.

For Mac users, you can use the program Automator, which is already installed on your computer. Please see this tutorial for step-by-step instructions, using the "Make Sequential" option - under "'new name", enter:
[yyyymmdd]_[last names (separated by dashes)]_[description (words separated by dashes)]
Then choose "underscore" as the symbol following the name; and set the number of digits to 3 (ie 001).

For PC users, download the "Windows standalone" version of Actions and see this tutorial for instructions, using the same settings as previously described for Automator.
